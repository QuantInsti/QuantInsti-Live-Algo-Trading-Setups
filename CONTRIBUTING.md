# Contributing to QuantInsti Live Algo Trading Setups

Thank you for your interest in contributing to this monorepo! This document provides general guidelines for you to contribute to any of the trading setups.

## Code of Conduct

This project and everyone participating in it is governed by our [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.

## How to Contribute

Due to the monorepo structure, each trading setup has its own specific contribution guidelines, development setup, and testing procedures. You should refer to the `CONTRIBUTING.md` file within the specific setup directory you are interested in:

-   [ibkr-crypto/CONTRIBUTING.md](ibkr-crypto/CONTRIBUTING.md)
-   [ibkr-forex/CONTRIBUTING.md](ibkr-forex/CONTRIBUTING.md)
-   [ibkr-stock/CONTRIBUTING.md](ibkr-stock/CONTRIBUTING.md)

## Reporting Issues

When you report issues, you should specify which trading setup the issue applies to.

## Questions?

If you have general questions about contributing to the monorepo, you should open an issue.
